- Testing framework: JUnit  

#### Các bước thực hiện
1. Sử dụng Java và Maven để thêm dependency JUnit.
2. Xây dựng lớp `StudentAnalyzer`.
3. Tạo lớp kiểm thử `StudentAnalyzerTest`.

5. Chạy các test case và kiểm tra kết quả.
6. Đẩy mã nguồn lên GitHub.
